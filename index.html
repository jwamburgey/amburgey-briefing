Configuration
App Setup
One-time setup — takes about 15 minutes
1
Claude API Key

Go to console.anthropic.com, create an account (same email is fine), then go to API Keys and create a new key. Paste it below.

Your daily sweep costs roughly $0.01–0.05. Usage is billed directly to your Anthropic Console account.

2
Google OAuth Client ID

Go to console.cloud.google.com → New Project → APIs & Services → Credentials → Create OAuth 2.0 Client ID (Web Application). Add file:// and your local file path as authorized origins. Paste your Client ID below.

3
Enable Google APIs

In Google Cloud Console → APIs & Services → Library, enable these three APIs:

Gmail API   Google Sheets API   Google Drive API

4
Google Sheet ID

Open your job tracker Google Sheet (or create a new one). The Sheet ID is in the URL: docs.google.com/spreadsheets/d/THIS_PART/edit. Paste it below.

The app will create the required tabs (GC Contacts, App State) automatically on first run if they don't exist.

5
Gmail Address

Confirm the Gmail address to scan for the morning sweep.

Keys stored locally in your browser only
